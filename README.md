# IoT Based Healthcare System

ESP32-based healthcare monitoring system for real-time acquisition and cloud transmission of vital health parameters including heart rate, ECG, SpO2, and body temperature.

## Features
- Real-time heart rate monitoring
- ECG signal acquisition
- SpO2 monitoring
- Temperature sensing
- OLED-based local display
- ThingSpeak cloud integration
- Emergency alert indication

## Hardware Used
- ESP32
- MAX30100 Pulse Oximeter
- ECG Sensor
- Temperature Sensor
- OLED Display
- WiFi Module

## Applications
- Remote patient monitoring
- IoT healthcare systems
- Wearable health monitoring
